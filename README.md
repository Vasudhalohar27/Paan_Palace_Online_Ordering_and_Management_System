# 🛒 Paan Palace Online Ordering & Management System

## 📌 Project Description

**Paan Palace** is a full-stack e-commerce style web application designed for selling and managing paan online.

This project follows a **monorepo architecture**, containing:

* Backend (Spring Boot API)
* User Interface (Customer side)
* Admin Panel (React-based dashboard)

---

## 🎯 Project Overview

### 👤 User Side

* Browse paan products
* Add items to cart
* Place orders
* Checkout functionality

### 👨‍💼 Admin Side

* Manage orders
* Manage customers
* Manage ingredients
* Dashboard built using React

### ⚙️ Backend

* REST API using Spring Boot
* JWT-based authentication
* Product & order management

## 🛠️ How to Run

### 🔹 Backend

```bash
cd paan-backend
mvn spring-boot:run
```

(or)

```bash
mvnw spring-boot:run
```

---

### 🔹 User Interface

```bash
cd user-interface
```

👉 Open `index.html` in browser
OR

```bash
npx serve .
```

---

### 🔹 React Admin Panel

```bash
cd react-admin
npm install
npm start
```
